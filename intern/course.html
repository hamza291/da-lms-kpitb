📅 Weeks 1–2
Phase 1 · Foundations

Foundations & Analytical Mindset

Understand the types of data analytics, the lifecycle of data, how metrics differ from KPIs, and the principles of privacy and ethics.

1.1 — What is Data Analytics?

⌄

Data analytics is the process of examining raw data to find patterns and answer questions. It's classified into four stages:

Descriptive

What happened?
e.g. Monthly sales report

Diagnostic

Why did it happen?
e.g. Analyzing a sales drop

Predictive

What will happen?
e.g. Predicting customer churn

Prescriptive

What should we do?
e.g. Route optimization

1.2 — The Data Lifecycle

⌄

Every analysis project follows these six distinct steps:

1. Collect -> Gather raw database values, logs, or API responses 2. Clean -> Handle nulls, format fields, remove duplicates (80% of time) 3. Explore -> Run exploratory data analysis (EDA) to find initial trends 4. Analyze -> Run statistical tests, models, or aggregations 5. Visualize -> Build dashboards (Power BI, Seaborn) to show results 6. Decide -> Present findings so managers can take action

1.3 — KPIs vs Metrics

⌄

A metric is any number you can track. A KPI (Key Performance Indicator) is a metric directly tied to a main business goal.

SMART PropertyMeaning
SpecificClearly defined parameter
MeasurableQuantified with a number
AchievableRealistic goal
RelevantSupports company success
Time-boundMeasured in a set duration (e.g. "by Q4")

1.4 — Data Types (NOIR Scale)

⌄

Understanding variables on the NOIR Scale:

ScaleTypeExample
NominalCategorical, no orderCity, Color, Gender
OrdinalCategorical with orderCustomer Rating (1-5 Stars)
IntervalNumeric, equal intervals, no true zeroTemperature in Celsius
RatioNumeric with a true zeroRevenue, Height, Weight

1.5 — Data Ethics

⌄

Ethics is critical for data analysts:

Privacy

Protect PII and customer identity.

Consent

Users must agree to data collection.

Bias

Biased data leads to discriminatory models.

Phase 2 · Business Domain

Business Domain Knowledge

Understand the metrics, vocabulary, and objectives of key business departments: Finance, Marketing, Operations, and HR.

2.1 — Finance & Accounting

⌄

Financial statements and core formulas:

Gross Profit = Revenue – COGS Net Income = Operating Income – Interest & Tax CAC = Total Marketing Spend / New Customers LTV = Avg Purchase Value × Frequency × Lifespan

2.2 — Marketing & Operations

⌄

Funnel conversion and efficiency analysis:

CTR = (Clicks / Impressions) × 100 ROAS = Revenue from Ads / Ad Spend Inventory Turnover = COGS / Average Inventory Defect Rate = (Defective Units / Total Units) × 100

Phase 3 · Statistics

Statistics & Data Analysis Basics

Learn descriptive statistics, probability theory, hypothesis testing rules, and data quality standards.

3.1 — Descriptive Statistics

⌄

Measures of center and dispersion:

Mean = Sum of values / Count of values Median = Middle value (unsusceptible to outliers) IQR = Q3 - Q1 (Interquartile range) Outlier Fences = [Q1 - 1.5*IQR, Q3 + 1.5*IQR]

3.2 — Hypothesis Testing

⌄

Evaluating statistical statements:

H0 (Null): No difference/effect H1 (Alternative): There is a significant difference/effect p-value < 0.05 -> Reject H0 (Result is statistically significant)

Phase 5 · SQL

SQL for Analytics

Query relational databases using SELECT, Joins, subqueries, CTEs, and window functions.

5.1 — Core SQL Queries

⌄
SELECT Dept, AVG(Salary) FROM Employees WHERE HireDate > '2025-01-01' GROUP BY Dept HAVING AVG(Salary) > 60000;

5.2 — Advanced SQL

⌄
WITH SalesCTE AS ( SELECT EmployeeID, SUM(Amount) AS Total FROM Sales GROUP BY EmployeeID ) SELECT Name, Total, RANK() OVER (ORDER BY Total DESC) AS SalesRank FROM Employees e JOIN SalesCTE s ON e.EmployeeID = s.EmployeeID;

Phase 7 · Visualization

Data Visualization & Storytelling

Learn dashboard design principles, color theory, Power BI development, and structural storytelling (SCQA).

7.1 — Design Principles

⌄
Data-Ink Ratio

Maximize actual data indicators. Remove gridlines and borders.

5-Second Rule

Viewers must grasp the main chart insight in 5 seconds.

Color Theory

Avoid rainbow colors. Use one accent color + grey for neutral data.
